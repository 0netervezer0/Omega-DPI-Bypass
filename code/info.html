Lupi DPI Launcher User Guide

Key points

  • The program executable file must be located in the same folder where the DPI folder is located.
  • The program requires administrator rights because it interacts with the network and the system firewall.
  • This version is only applicable to Windows 7 and above.
  • The program is not a VPN service, it does not affect the IP address of your device in any way.

Scheme of work with the program

  • The "Launch DPI Bypass" button will launch the winws.exe subroutine, which will open as a terminal. You cannot close the terminal, otherwise it will interrupt the program.
  • After opening the terminal, the main program window can be closed. To stop the program, use the "Stop DPI Bypass" button in the main menu or simply close the terminal.
  • The main program menu also includes the "Open DPI Folder" and "Open Address List" buttons for working with files and address list.
  • The "Install Bypass Service" button will install the service, namely, it will start the DPI bypass service in the background process and add it to your system startup, after installing the service you will no longer need to manually start the DPI bypass.
  • The "Remove Bypass Service" button will remove the previously installed service, this must be done if you are going to reinstall the installed service.
